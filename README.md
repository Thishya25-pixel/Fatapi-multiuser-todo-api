# FastAPI Multi-User Todo App

## Features
- User Signup & Login
- JWT Authentication
- Secure Multi-User Todo Management
- CRUD Operations
- SQLite + SQLAlchemy ORM

## Tech Stack
- FastAPI
- SQLAlchemy
- SQLite
- JWT (python-jose)
- Passlib (bcrypt)

## How to Run

1. Install dependencies
   pip install -r requirements.txt

2. Run server
   uvicorn main:app --reload

3. Open docs
   http://127.0.0.1:8000/docs# Fatapi-multiuser-todo-api
